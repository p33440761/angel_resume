我帶來的流量 - IG

面對Instagram觸及率低迷，我展現敏銳的數位洞察力與創意策劃能力，向主管建議採用「圖文」形式，挑選受眾喜愛的主題，將記者稿件整合為吸引人的小編文案，並精準下標以提升點擊率。此策略成功提升觸及數，展現了我善於分析數據、優化內容並快速應對社群趨勢的能力。




我參與了「食尚玩家手搖祭」專案的社群宣傳，運用美編能力設計一組活潑的宣傳圖組，搭配吸引人的文案，精準抓住受眾興趣，有效提升社群互動率與專案曝光度。我擁有整合視覺與文字的內容創作能力，也彰顯了高效的跨部門協作技巧，成功推動該專案。




針對食尚玩家年齡層受眾普遍偏高，精心挑選符合其喜好的主題，設計吸睛的社群輪播圖，並整合記者新聞稿的精華文字，展現簡潔且具吸引力的內容。
